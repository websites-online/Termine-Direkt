So funktioniert’s

In drei Schritten zur fertigen Online-Reservierung.

01

Link erhalten

Sie bekommen eine eigene Reservierungsseite, z. B. /r/max-grill

02

Bei Google eintragen

Der Link wird im Google-Unternehmensprofil hinterlegt.

03

Reservierungen erhalten

Gäste buchen online – Sie erhalten sofort eine E-Mail.
